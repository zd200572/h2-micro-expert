# Hydrogenase Classification Summary (HydDB-based)

This reference defines the physiological direction (Evolving vs. Uptake) based on HydDB 2.0 and Greening et al. (2016).

## [NiFe]-Hydrogenases (4 Groups)

| Group | Subtype | Primary Function | Physiological Direction | Key Species Example |
| :--- | :--- | :--- | :--- | :--- |
| **Group 1** | 1a-1h | Respiratory H2 uptake | **Uptake (Consumption)** | *Ralstonia eutropha* |
| **Group 2** | 2a-2e | H2 sensing or cyanobacterial uptake | **Uptake / Regulatory** | *Anabaena variabilis* |
| **Group 3** | 3a-3d | Bidirectional / Cytoplasmic | **Bidirectional** | *Pyrococcus furiosus* |
| **Group 4** | 4a-4g | Energy-converting (H+ reduction) | **Evolving (Production)** | *Escherichia coli* (Hyd-3/4) |

## [FeFe]-Hydrogenases (4 Groups)

| Group | Subtype | Primary Function | Physiological Direction | Key Species Example |
| :--- | :--- | :--- | :--- | :--- |
| **Group A** | A1-A4 | Fermentative H2 production | **Evolving (Production)** | *Clostridium butyricum* |
| **Group B** | B1-B2 | Putative bifurcating or uptake | **Variable** | *Thermotoga maritima* |
| **Group C** | C1-C3 | Sensory or regulatory | **Regulatory** | *Chlamydomonas reinhardtii* |
| **Group D** | D | Unknown / Novel | **Unknown** | Uncultured archaea |

## [Fe]-Hydrogenases (Hmd)
- Found in methanogenic archaea.
- Function: CO2 reduction with H2.
- **Direction: Uptake (Consumption)**.
